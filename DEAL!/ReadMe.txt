DEAL! is an application that is based on gamification of reminder application like such as Habitica, etc. Created by 6 amazing creators, DEAL! brings to you a reminder application that is not only functional but also very fun and interactive to use. You can check the app version in our Itch.io page, https://ouroboroz-studios.itch.io/deal

CONTROLS:
- Add / Edit Panel
[Tab] - Switch between input fields.

- Note Board
[M1] - Create a new note and drag it to move.

- Double Left Click 
[M1] x2 - Edit the selected note.

- Right Click 
[M2] - Release the dragged note.

- Double Right Click 
[M2] x2 - Delete the selected note.

- Middle Scroll
[M3] - Change the note color.
